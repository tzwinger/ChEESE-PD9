# ChEESE-PD9
Reposiotry for ChEESE-2P Pilot Demonstrator 9. This PD is about testing the functionalisites of ELmer/Ice to simulate rapidly rising Glacier Lake Outburst Floods (GLOFs) under glaciers, better known as jökulhlaups.

## Contents
The repository consists of two test-cases (9.1 and 9.2)
* test-case 9.1: [Artificially triggered jökuhlaup in Arctic glacier](MLB_hydro)
* test-case 9.2: [Water transfer from a filled cauldron in synthetic setup resembling SC9.1](Chauldron)

See the README-files in the spcific subdirectories for in-depth instructions
